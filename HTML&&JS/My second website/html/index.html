近日，国务院办公厅印发《关于推进医疗联合体建设和发展的指导意见》。意见提出，开展医疗联合体（以下简称医联体）建设，是深化医改的重要步骤和制度创新，有利于调整优化医疗资源结构布局，促进医疗卫生工作重心下移和资源下沉，提升基层服务能力，有利于医疗资源上下贯通，提升医疗服务体系整体效能，更好实施分级诊疗和满足群众健康需求。 　　4月20日，李克强总理在考察威海孙家疃医院时就强调了医联体的重要作用：中医讲“通则不痛”。我们就是要让优质医疗资源上下贯通，以医联体之“通”，破解群众看病之“痛”。作为社区医院的威海孙家疃医院与威海市中医院组建医联体，市中医院专家可以在此出诊并视频远程诊疗。 　　此次指导意见所言明的医联体的重要性不言而喻，也对医联体的功能与目标进一步做了明确。医联体最重要的目的是要将优质医疗资源“下沉”，要求试点省份的三级公立医院要全部参与并发挥引领作用，每个地市以及分级诊疗试点城市至少建成一个有明显成效的医联体。同时，还需要探索多种形式的医联体，除了地市级医院与基层、社区医院的联合，还要求县级区域内也进行医联。李克强总理用中药治病的原理来比喻医联体，非常形象与恰当。原来，在一定区域内，病人的检验结果医院互相不认可，处方缺乏流动性，药品也很难共享，病人转诊机制非常不畅。更重要的，医保支付缺乏统筹令许多病人不便。 　　区域内医疗资源的“不通”成为一种循环，并且还有着自我强化的特征。对于吸纳了各种优质资源的三级医院们来说，它们垄断资源、“跑马圈地”、“虹吸”基层资源、挤压社会办医空间，但同时这个负荷过重的机构又无法为每个病人提供很好的服务。公立医院改革是新医改中最重要最难啃的一部分，破除以药养医、建立现代医院制度正在推进，医联体的建设就是要推动它们愿意将一部分医疗资源下沉到基层，起到带动作用。这同时也是在为大医院们减负，以使得它们轻装上阵，将更多精力放在疑难重症与科研方面，并且更加体现公益性。 　　此次医联体试点也要求建立明确的绩效考核机制，包括建立医联体考核指标体系，重点考核医联体技术辐射带动情况、医疗资源下沉情况等。也会将考核评价结果作为人事任免、评优评先等的重要依据，并与医务人员绩效工资、进修、晋升等挂钩。在经过试点之后，医联体的改革时间表也非常明确，计划在2020年全面实施，而在试点阶段需要完成的改革任务非常繁重，包括促进公立医院改革、医保支付方式改革、分级诊疗制度建设等。
